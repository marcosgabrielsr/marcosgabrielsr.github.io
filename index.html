Bomb Drop - The Game

This project is a game based on Pou's Fruit Drop made with the Arduino framework. In this game, you must prevent bombs from passing the player-controlled platform, while also avoiding falling anchors. Each bomb caught by the platform earns the player a point; they lose a life if a bomb passes through the platform or if an anchor hits the platform.

Random Math - The Game

This project is a game where you have to solve as many simple mathematical equations as possible within a specified time limit, made with the Arduino framework. For each solved equation, you earn one point, and currently, there is no penalty for each mistake.
